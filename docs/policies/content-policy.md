# Content Policy

This repository exists to share knowledge, not copyrighted full-text translations.

## Allowed

- glossary mappings;
- prompt templates;
- workflow advice;
- short example phrases used for explanation;
- commentary about terminology and readability.

## Not Allowed

- full translated chapters from copyrighted novels;
- large copied source excerpts;
- scraped premium content;
- unauthorized bilingual dumps.

## Practical Rule

If a contribution mainly helps readers learn a method, it is likely acceptable.

If a contribution mainly republishes the book itself, it should not be in this repository.
