# Xiuxian / Cultivation Novel Reading Guide

"Xiuxian" is often used broadly by readers to describe cultivation-focused web novels, including many stories adjacent to xianxia or progression fantasy.

## What Makes Xiuxian Novels Difficult For MTL

- long power ladders;
- repeated system terminology;
- item, technique, and inheritance naming drift;
- many near-synonymous metaphysical terms.

## Best Workflow

For cultivation-heavy novels:

1. map the full realm ladder as early as possible;
2. keep a live glossary for techniques and treasures;
3. preserve recurring metaphysical vocabulary;
4. reject outputs that rename established systems.

## Terms Worth Preserving Carefully

- qi;
- Dao;
- dantian;
- spiritual root;
- heavenly tribulation;
- core formation or Golden Core style milestones.

## Practical Rule

If the novel is system-heavy, you are not just translating prose. You are maintaining a terminology framework.
