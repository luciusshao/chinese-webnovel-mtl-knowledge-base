# Name Translation Strategies

There is no single correct approach for names. The important thing is to choose a strategy and keep it consistent.

## Common Approaches

### 1. Full Transliteration

Example:

- 林澈 -> Lin Che
- 青岚宗 -> Qinglan Sect

Best when:

- preserving Chinese flavor matters;
- names are proper nouns rather than descriptive titles.

### 2. Meaning Translation

Example:

- 青岚宗 -> Azure Mist Sect

Best when:

- the name meaning matters for tone or symbolism;
- the audience prefers more transparent English.

### 3. Mixed Style

Example:

- 青岚宗 -> Qinglan Sect
- 天剑峰 -> Heavenly Sword Peak

Best when:

- you want a balance between readability and cultural texture.

## Recommended Default

For public guidance, mixed style is often the most robust default.

## Bad Outcome To Avoid

The same name should not alternate between transliteration and meaning-translation unless there is a strong reason.
