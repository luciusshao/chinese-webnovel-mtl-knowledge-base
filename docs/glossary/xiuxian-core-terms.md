# Xiuxian / Cultivation Core Terms

This page is a broader cultivation-oriented starter glossary for readers who use "xiuxian" as an umbrella category.

| Chinese | Preferred English | Alternatives | Notes |
|---|---|---|---|
| 修仙 | cultivation | immortal cultivation | broad umbrella term |
| 修士 | cultivator | cultivation practitioner | most common default is cultivator |
| 灵石 | spirit stone | spiritual stone | lock one |
| 功法 | cultivation method | cultivation art | highly context-sensitive |
| 心法 | mental method | inner cultivation method | often paired with 功法 |
| 境界 | realm | stage | usually use realm |
| 突破 | breakthrough | advance | important progression term |
| 闭关 | closed-door cultivation | seclusion cultivation | keep stable |
| 本命法宝 | life-bound treasure | natal treasure | choose one style |
| 传承 | inheritance | legacy | lore-dependent |
