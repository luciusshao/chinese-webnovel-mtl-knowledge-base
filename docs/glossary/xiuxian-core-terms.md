# Xiuxian / Cultivation Core Terms

This page is a broader cultivation-oriented starter glossary for readers who use "xiuxian" as an umbrella category.

| Chinese | Preferred English | Notes |
| --- | --- | --- |
| 修仙 | cultivation | broad umbrella term |
| 修士 | cultivator | most common default is cultivator |
| 灵石 | spirit stone | lock one |
| 功法 | cultivation method | highly context-sensitive |
| 心法 | mental method | often paired with 功法 |
| 境界 | realm | usually use realm |
| 突破 | breakthrough | important progression term |
| 闭关 | closed-door cultivation | keep stable |
| 本命法宝 | life-bound treasure | choose one style |
| 传承 | inheritance | lore-dependent |
