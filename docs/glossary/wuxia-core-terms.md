# Wuxia Core Terms

This page provides a starter set of wuxia-oriented terminology.

| Chinese | Preferred English | Notes |
| --- | --- | --- |
| 江湖 | jianghu | often better kept as jianghu |
| 武林 | martial world | depends on tone |
| 掌门 | sect leader | lock one version |
| 长老 | elder | hierarchy may vary |
| 内力 | inner force | style choice |
| 轻功 | lightness skill | both are common |
| 剑法 | sword art | depends on naming style |
| 刀法 | blade art | keep style aligned |
| 暗器 | hidden weapon | common wuxia term |
| 武学 | martial art | broad category term |
