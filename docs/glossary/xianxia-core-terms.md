# Xianxia Core Terms

This page provides a starter set of xianxia-oriented terminology.

| Chinese | Preferred English | Alternatives | Notes |
|---|---|---|---|
| 灵气 | spiritual qi | qi | preserve genre feel |
| 道心 | Dao heart | heart of Dao | keep stable |
| 丹田 | dantian | elixir field | transliteration often reads better |
| 法宝 | magical treasure | magic treasure | common category term |
| 洞府 | cave residence | cave abode | depends on tone |
| 机缘 | fortuitous encounter | lucky chance | common xianxia term |
| 炼丹 | alchemy | pill refining | context-sensitive |
| 灵根 | spiritual root | spirit root | very common cultivation term |
| 天劫 | heavenly tribulation | celestial tribulation | lock one form |
| 飞升 | ascension | transcendence | depends on novel cosmology |
