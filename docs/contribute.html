What happens next

  1. Each row goes to a private review queue.
  2. The maintainer reviews entries — usually within a few days.
  3. Approved terms are merged into the public glossary CSV files.
  4. If a term you submit already exists, the maintainer either folds your reason into the existing entry or skips it.
  5. Spam, copyrighted content, or off-topic submissions are dropped silently.

Want to see what's already collected before contributing? Use the glossary browser or grab the CSV files.
