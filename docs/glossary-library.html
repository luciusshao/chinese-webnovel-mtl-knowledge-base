Glossary library

Ready-to-use CSV files for ordinary readers. Open them in Excel or Numbers, paste the rows into your AI translator, or feed them to your own glossary tool. No GitHub account needed.

Want to pick and choose?

The online glossary browser lets you search, filter by genre, and tick the rows you want — then export your selection as CSV or Markdown. Useful when you only care about a specific novel or character.

Suggested columns

The CSV files use the same four columns so non-technical users can edit them easily:

  • genre — xianxia, wuxia, xiuxian, or other
  • chinese — the Chinese term as it appears in source novels
  • preferred_english — the recommended English rendering
  • notes — short reasoning, e.g. "transliteration reads better here"

How to contribute back

If you improve a glossary file: download the CSV, edit rows, and submit through the contribute page — either via the online form or by emailing the file to the maintainer. No GitHub required.
