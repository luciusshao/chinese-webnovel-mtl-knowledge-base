# Basic Chapter Translation Prompt

Use this as a starting point for readable, consistent chapter translation.

```text
Translate this Chinese web novel chapter into readable English.

Requirements:
1. Follow the glossary exactly.
2. Keep all character names consistent.
3. Keep cultivation realms and technical terms consistent.
4. Do not summarize, skip, or compress content.
5. Preserve the serialized web novel tone.
6. Do not replace Chinese fantasy concepts with generic Western fantasy terms unless the glossary says so.

Glossary:
[paste glossary here]

Chapter text:
[paste chapter here]
```

## When To Customize

Add extra rules when you need:

- more literal output;
- smoother English prose;
- stricter dialogue formatting;
- retained pinyin for specific concepts.
