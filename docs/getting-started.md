# Getting Started

If you are new to reading Chinese web novels with MTL, start with a simple workflow.

## Minimal Workflow

1. Pick one novel and one translation setup.
2. Build a tiny glossary before reading too far.
3. Lock names and cultivation realms early.
4. Use one prompt template consistently.
5. Update the glossary whenever repeated terms drift.

## First Glossary Checklist

At minimum, collect:

- main character names;
- sect and clan names;
- realm names;
- skill and technique names;
- recurring items and places.

## Success Criteria

Your MTL setup is working if:

- names stop changing every few chapters;
- cultivation stages stay consistent;
- tone remains readable for long-form reading;
- you spend less time mentally correcting basic terms.
