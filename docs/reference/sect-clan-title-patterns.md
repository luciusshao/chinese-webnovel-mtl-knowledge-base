# Sect, Clan, And Title Patterns

These patterns help keep naming decisions consistent across novels.

## Common Structural Terms

| Chinese | Common English |
|---|---|
| 宗 | Sect |
| 门 | Sect / School |
| 派 | Sect / School |
| 宫 | Palace |
| 殿 | Hall |
| 阁 | Pavilion |
| 楼 | Tower |
| 城 | City |
| 国 | Kingdom / Nation |
| 朝 | Dynasty |
| 家 | Clan / Family |

## Common Role Titles

| Chinese | Common English |
|---|---|
| 宗主 | Sect Master |
| 长老 | Elder |
| 掌门 | Sect Leader / Sect Master |
| 峰主 | Peak Master |
| 圣子 | Holy Son / Saint Son |
| 圣女 | Holy Maiden / Saintess |

## Practical Rule

Pick one rendering for each structural suffix and apply it consistently inside the same project.
