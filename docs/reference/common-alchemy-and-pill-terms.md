# Common Alchemy And Pill Terms

Alchemy terminology is highly repetitive in cultivation novels and should be standardized early.

| Chinese | Common English | Alternatives | Notes |
|---|---|---|---|
| 炼丹 | pill refining | alchemy | broad action term |
| 丹药 | pill | medicinal pill | common default is pill |
| 灵丹 | spiritual pill | spirit pill | item rank/tone varies |
| 药材 | medicinal ingredient | herbs and materials | broad ingredient term |
| 药鼎 | pill cauldron | alchemy cauldron | common vessel term |
| 丹火 | pill flame | alchemical flame | context-sensitive |
| 丹毒 | pill toxin | pill impurity | depends on system |
| 药效 | medicinal effect | effect | common evaluation term |
| 成丹 | complete the pill | pill formation | context-sensitive process term |
| 废丹 | ruined pill | failed pill | failure state |

## Practical Rule

If the novel has pill grades, record the full grade ladder along with the item terms.
