# Common Alchemy And Pill Terms

Alchemy terminology is highly repetitive in cultivation novels and should be standardized early.

| Chinese | Common English | Notes |
| --- | --- | --- |
| 炼丹 | pill refining | broad action term |
| 丹药 | pill | common default is pill |
| 灵丹 | spiritual pill | item rank/tone varies |
| 药材 | medicinal ingredient | broad ingredient term |
| 药鼎 | pill cauldron | common vessel term |
| 丹火 | pill flame | context-sensitive |
| 丹毒 | pill toxin | depends on system |
| 药效 | medicinal effect | common evaluation term |
| 成丹 | complete the pill | context-sensitive process term |
| 废丹 | ruined pill | failure state |

## Practical Rule

If the novel has pill grades, record the full grade ladder along with the item terms.
