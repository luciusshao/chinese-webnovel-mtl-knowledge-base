# Common Ranks And Status Terms

Ranks, talent labels, and status terms appear everywhere in progression and martial stories.

| Chinese | Common English | Notes |
| --- | --- | --- |
| 天才 | genius | common talent label |
| 妖孽 | monster | often metaphorical praise |
| 废物 | trash | insult term, tone-sensitive |
| 外门弟子 | outer disciple | sect hierarchy term |
| 内门弟子 | inner disciple | sect hierarchy term |
| 核心弟子 | core disciple | common rank term |
| 亲传弟子 | direct disciple | lineage-important term |
| 候选人 | candidate | context-sensitive |
| 天骄 | favored heaven talent | difficult prestige term |
| 圣子 | Holy Son | lock one style |
| 圣女 | Holy Maiden | lock one style |

## Practical Rule

Rank labels often encode social structure. Treat them as system terms, not decorative wording.
