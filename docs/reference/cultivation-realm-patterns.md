# Cultivation Realm Patterns

Cultivation systems vary by novel, but many share overlapping terminology.

## Common Ladder Example

| Chinese | Common English |
|---|---|
| 炼气 | Qi Refining |
| 筑基 | Foundation Establishment |
| 金丹 | Golden Core |
| 元婴 | Nascent Soul |
| 化神 | Spirit Transformation |

## Rule For Contributors

If a novel uses a custom realm system, document the whole ladder rather than isolated terms.

## Why It Matters

Readers tolerate awkward prose more than unstable power-system terminology.
