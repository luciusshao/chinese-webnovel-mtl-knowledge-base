# Common Action And Cultivation Verbs

These verbs appear constantly in cultivation and martial stories. If they drift, readability drops quickly.

| Chinese | Common English | Notes |
| --- | --- | --- |
| 修炼 | cultivate | cultivation stories usually prefer cultivate |
| 突破 | break through | key progression verb |
| 闭关 | enter seclusion | often fixed phrasing |
| 炼化 | refine | item or energy context matters |
| 吞噬 | devour | tone varies by novel |
| 感悟 | comprehend | common Dao-context verb |
| 领悟 | realize | often paired with laws/intent |
| 镇压 | suppress | power gap verb |
| 斩杀 | slay | elevated action term |
| 驾驭 | control | mount, treasure, or energy context |

## Practical Rule

High-frequency verbs need a stable style just as much as nouns do.
