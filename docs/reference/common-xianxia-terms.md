# Common Xianxia Terms

This page is a starter reference, not a universal standard.

| Chinese | Common English | Notes |
|---|---|---|
| 灵气 | spiritual qi | sometimes left as qi |
| 道心 | Dao heart | often better than over-explaining |
| 丹田 | dantian | usually keep transliterated |
| 法宝 | magical treasure | context may vary |
| 功法 | cultivation method | can also be technique/manual depending on context |
| 洞府 | cave residence | sometimes immortal cave abode |
| 机缘 | fortuitous encounter | common progression term |
| 炼气 | Qi Refining | lock early |
| 筑基 | Foundation Establishment | lock early |
| 金丹 | Golden Core | very common standard |
| 元婴 | Nascent Soul | very common standard |
