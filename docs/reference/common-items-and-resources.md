# Common Items And Resources

Items, pills, stones, manuals, and resources recur across long novels and should be standardized early.

| Chinese | Common English | Alternatives | Notes |
|---|---|---|---|
| 灵石 | spirit stone | spiritual stone | very common resource term |
| 丹药 | pill | medicinal pill | cultivation context matters |
| 秘籍 | secret manual | manual | martial/cultivation text |
| 玉简 | jade slip | jade tablet | often use jade slip |
| 储物袋 | storage bag | holding bag | common inventory item |
| 储物戒 | storage ring | spatial ring | both are common |
| 法器 | magic artifact | artifact | rank below 法宝 in some settings |
| 阵法 | formation | array formation | context-sensitive |
| 符箓 | talisman | rune talisman | common in Daoist fantasy |
| 灵药 | spiritual herb | spirit herb | medicine/alchemy context |

## Practical Rule

Track rank differences between item categories if the novel treats them as mechanically distinct.
