# Common Items And Resources

Items, pills, stones, manuals, and resources recur across long novels and should be standardized early.

| Chinese | Common English | Notes |
| --- | --- | --- |
| 灵石 | spirit stone | very common resource term |
| 丹药 | pill | cultivation context matters |
| 秘籍 | secret manual | martial/cultivation text |
| 玉简 | jade slip | often use jade slip |
| 储物袋 | storage bag | common inventory item |
| 储物戒 | storage ring | both are common |
| 法器 | magic artifact | rank below 法宝 in some settings |
| 阵法 | formation | context-sensitive |
| 符箓 | talisman | common in Daoist fantasy |
| 灵药 | spiritual herb | medicine/alchemy context |

## Practical Rule

Track rank differences between item categories if the novel treats them as mechanically distinct.
