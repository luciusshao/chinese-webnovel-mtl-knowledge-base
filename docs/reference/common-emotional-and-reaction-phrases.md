# Common Emotional And Reaction Phrases

These phrases show up constantly in serialized fiction and often sound unnatural in raw MTL.

| Chinese | Common English | Notes |
| --- | --- | --- |
| 冷哼一声 | snorted coldly | very common narration phrase |
| 脸色一变 | his expression changed | smoother English preferred |
| 倒吸一口凉气 | sucked in a cold breath | common shock reaction |
| 心中一惊 | was startled inwardly | internal reaction phrase |
| 杀意 | killing intent | both common |
| 气息 | aura | cultivation context matters |
| 威压 | pressure | common xianxia force term |
| 神色复杂 | expression complicated | often needs smoothing |
| 目瞪口呆 | dumbfounded | common reaction phrase |
| 若有所思 | thoughtful | repeated prose pattern |

## Practical Rule

Phrase-level consistency matters because serialized novels reuse the same reactions heavily.
