# Glossary Library

This page is for ordinary users who want to download, reuse, or submit glossary files without learning GitHub workflows.

## Download Ready-To-Use Glossaries

- [Xianxia Core Terms CSV](downloads/xianxia-core-terms.csv)
- [Wuxia Core Terms CSV](downloads/wuxia-core-terms.csv)
- [Xiuxian Core Terms CSV](downloads/xiuxian-core-terms.csv)
- [Starter Combined Glossary CSV](downloads/starter-combined-glossary.csv)

## How To Use These Files

You can:

- open them in Excel or Numbers;
- paste them into your own glossary tool;
- copy selected terms into an AI translation prompt;
- adapt them for your own novel reading workflow.

## Suggested Columns

These glossary files use simple columns so non-technical users can edit them easily:

- `genre`
- `chinese`
- `preferred_english`
- `alternatives`
- `notes`

## How To Contribute Back

If you improve a glossary file, you can:

1. download the CSV;
2. edit the rows;
3. save your version;
4. send it back to the maintainer.

You do not need GitHub for this path.

## Best Use Case

This works well for readers who want to share 5 to 50 repeated terms from one genre or one novel.
