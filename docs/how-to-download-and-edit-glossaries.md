# How To Download And Edit Glossaries

You do not need a coding setup to use the glossary files.

## Simple Workflow

1. open the [Glossary Library](glossary-library.md);
2. download a CSV file;
3. open it in Excel, Numbers, Google Sheets, or any spreadsheet app;
4. edit the English terms or notes;
5. save your copy;
6. send the updated file back to the maintainer if you want to contribute it.

## What To Edit First

Start with:

- names that keep drifting;
- cultivation realm ladders;
- sect, clan, and title names;
- repeated technical terms.

## Good Editing Rule

Do not try to perfect every term at once. Fix the terms that appear many times first.

## If You Only Have A Phone

Use the text templates instead of CSV, or open the CSV in Google Sheets mobile and change only the most important rows.
