# Contribute Without GitHub

You do not need to know GitHub to help improve this glossary library.

## Fastest Way

1. Open one of the templates below.
2. Copy it.
3. Fill in the terms you know.
4. Send it to the repository maintainer in whatever channel is easiest.

Recommended templates:

- [Glossary Submission Template](forms/glossary-submission-template.md)
- [Quick Term Submission Template](forms/quick-term-submission-template.md)

## What To Send

Useful contributions include:

- one Chinese term and a better English rendering;
- a short list of cultivation realms;
- sect, clan, or title naming suggestions;
- repeated terms from a genre like xianxia or wuxia.

## Good Example

```text
Genre: xianxia
Chinese term: 天劫
Preferred English: heavenly tribulation
Avoid: sky calamity
Reason: more standard and easier for readers to recognize
```

## Best Kind Of Contribution

The best contributions are:

- short;
- concrete;
- consistent;
- based on repeated terms, not one-off wording.

## Important Rule

Please do not send full copyrighted translated chapters. Send terminology, glossary suggestions, and short examples only.
