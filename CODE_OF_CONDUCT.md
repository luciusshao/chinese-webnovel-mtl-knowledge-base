# Code Of Conduct

This repository is a practical public knowledge base. Keep the project useful, calm, and easy to collaborate on.

## Expected Behavior

- discuss terminology in good faith;
- explain preferences with reasons;
- accept that some translation choices are subjective;
- focus on usefulness for readers and contributors.

## Unacceptable Behavior

- harassment or personal attacks;
- ideological fights unrelated to the repository;
- spam, promotion, or link dumping;
- posting pirated or copyrighted full-text content.

## Maintainer Standard

Maintainers may remove content that is hostile, low-signal, or legally risky.
