# Chinese Webnovel MTL Knowledge Base

An open, community-contributed knowledge base for making machine-translated Chinese web novels more readable.

This repository is meant to be public, easy to browse, and easy to contribute to. The goal is not to host copyrighted chapter translations. The goal is to collect reusable reading workflows, terminology guidance, prompt patterns, glossary conventions, and legal-safety practices for readers, hobbyists, and tool builders.

## What This Repo Is For

- Practical advice for reading Chinese web novels with MTL
- Shared glossary conventions for common xianxia, wuxia, and fantasy terms
- Prompt patterns that improve consistency and readability
- Notes on common MTL failure modes
- Public-reference material that anyone can improve through pull requests

## What This Repo Is Not For

- Reposting full copyrighted translated chapters
- Hosting scraped paid content
- Publishing unauthorized full-novel English versions
- Sharing private API keys, prompts with secrets, or user data

## Contribution Model

Anyone should be able to contribute by:

- fixing wording;
- adding glossary entries;
- improving prompts;
- documenting genre-specific terminology;
- reporting inconsistent or confusing advice.

If you want this repo to stay broadly useful, keep content:

- generic rather than tool-locked;
- practical rather than promotional;
- safe from copyright problems.

## Suggested Next Contributions
